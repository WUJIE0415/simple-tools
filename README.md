# Simple Tools

一个本地运行的轻量级下载器网页工具。启动后会在本机浏览器中打开页面，用户可以粘贴公开可访问的媒体链接，并选择下载视频、导出音频或保存图片等内容。

本项目的定位是个人效率工具：界面尽量简单，运行尽量本地化，不提供云端托管、账号系统、远程存储或内容分发服务。

## 功能特性

- 本地 Web 界面，默认运行在 `http://127.0.0.1:4173`
- 支持粘贴链接后检测并下载媒体内容
- 支持视频下载
- 支持导出 `MP3`、`WAV` 音频
- 支持图片下载
- 支持 Instagram 公开图片、轮播、Reel 等内容的预览与批量下载
- 内置或调用 `yt-dlp`、`ffmpeg`、`ffprobe` 等工具完成媒体处理
- Windows 打包版本可直接运行，不需要用户额外安装 Node.js、Python、yt-dlp 或 ffmpeg

## 安装与使用

### 方式一：使用打包版本

1. 下载发布包并完整解压。
2. 保持文件夹结构不变，不要只复制单个 `VideoDownloader.exe`。
3. 双击运行 `VideoDownloader.exe`。
4. 程序会自动启动本地服务，并在默认浏览器中打开页面。
5. 在输入框中粘贴媒体链接。
6. 根据页面选项下载视频、音频或图片。

推荐目录结构：

```text
VideoDownloader/
  VideoDownloader.exe
  server.js
  index.html
  styles.css
  script.js
  assets/
  bin/
    yt-dlp.exe
    ffmpeg.exe
    ffprobe.exe
    instagram_provider.exe
```

### 方式二：从源码运行

适合开发者或需要自行调试的人。

```bash
npm install
npm start
```

启动后访问：

```text
http://127.0.0.1:4173
```

如果本地没有内置工具，请确保系统中可以正常调用 `yt-dlp` 和 `ffmpeg`，或将对应可执行文件放入 `bin/` 目录。

## 常见问题

### 页面打不开

请确认程序没有被 Windows SmartScreen、安全软件或防火墙拦截。也可以检查 `4173` 端口是否被其他程序占用。

### 提示找不到 yt-dlp 或 ffmpeg

请确认 `bin/` 目录中存在 `yt-dlp.exe`、`ffmpeg.exe` 和 `ffprobe.exe`，并且没有被安全软件删除或隔离。

### Instagram 下载失败

可能原因包括内容不是公开内容、链接需要登录、地区不可访问、内容已删除、网络异常，或 Instagram 页面结构变化导致解析工具暂时失效。

### 下载失败但链接能打开

部分平台会限制自动化下载，或要求登录、验证码、Cookie、地区访问权限。请确认你的使用方式符合平台规则。
